# Quadruple Filter Breakout

Packaged reference bundle for strict multi-timeframe trend gating plus breakout
timing.

Why this case is a strong fit:

- it matches the current `standard_bot` emphasis on multi-timeframe and
  breakout-friendly strategies
- it can be expressed as a clean combination of trend filters plus entry logic
- it is useful both for conservative trading and for teaching agents how to
  separate filters from triggers
