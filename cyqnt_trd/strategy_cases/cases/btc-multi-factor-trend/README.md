# BTC Multi-Factor Trend

Packaged reference bundle for a BTC-only multi-factor trend workflow.

Why this case is valuable:

- it is close to the current direction of `standard_bot`
- it combines price trend, momentum, derivatives, and volatility into one
  scoreable structure
- it is a strong blueprint for future case-to-plugin expansion

Current package stance:

- useful as a packaged factor bundle and preset template
- not yet one single executable plugin
- best treated as an orchestrated combination of existing and planned plugin
  families
