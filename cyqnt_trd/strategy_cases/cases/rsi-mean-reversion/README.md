# RSI Mean Reversion

Packaged reference bundle for the RSI mean-reversion family.

Why this case is packaged first:

- it already overlaps strongly with the current `rsi_reversion` plugin
- it is easy to explain to users and agents
- it provides a good template for oversold/overbought workflows and laddered
  variants

Current package stance:

- directly useful as a built-in case reference
- partially executable with the existing `rsi_reversion` plugin
- some variants still need dedicated implementation if you want the full
  original case behavior
